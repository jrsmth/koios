## MIT Problem : Difference of Squares
* Simplify this fraction:
    * $x^6 - a^2x^3y \over x^6 - a^4y^2$
    * $x^3(x^3 - a^2y) \over x^6 - a^4y^2$
    * $x^3(x^3 - a^2y) \over (x^3 + a^2y)(x^3 - a^2y)$
    * $x^3 \over x^3 + a^2y$
